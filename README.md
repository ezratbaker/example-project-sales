# example-project-sales
- In this example project, I analyze simulated grocery sales data from a database of 7 interconnected data tables. The data is available on Kaggle, at: https://www.kaggle.com/datasets/andrexibiza/grocery-sales-dataset
- I play the role of a company sales analyst helping the company identify top products, customers, and employees.
- The project features accessing data via API, preparing the database in SQLite, querying the database using SQL code, and analyzing and vizualing the data with Python commands.
